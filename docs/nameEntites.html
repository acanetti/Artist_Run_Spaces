DESCRIPTION
Sur ce graphe (ou réseau) chaque noeud représente un artist run space et chaque arc représente la similitude, d'après les entité entre chaque comité d'artiste .

Le degrés d'un noeuds est représenté par sa couleur, plus la nuance est foncé plus le degré du noeuds est élevé. Celà fait transparaître le nombre de commité relié à ce noeud.

La taille de chaque noeuds dépend de son coefficient de clustering, qui na vous donner un indicateur de sa capacité à relier les artist run space entre eux à travers le réseau .

PARAMETRES

Sur ce tableau vous pouvez tirer des informations spécifiques aux graphes choisi:
  • La taille du graphe : Nombre d'arêtes que possède le graphe, c'est à dire nombre de liaisons (le site A possède au minimum un mots clés en communs avec le site B) pour le cluster choisi.
  • Degré : Nombre d'arêtes que possède un sommet i, c'est à dire le nombre de mots clés en communs entre un site et les autres (exemple : si le sommet A possède 4 arêtes, son degré sera égal à 4. Parmi ces mots clés 4 d'entre eux sont utilisés aussi par d'autre site artistique).
  • Degré minimum : Le site (ou sommet) qui a le moins de mots clés en communs avec les autres sites.
  • Degré maximum : Le site (ou sommet) qui possède le plus de mot clés en commun avec les autres sites.
  • Degré moyen : En moyenne combien de mots clés en communs possède les sites entre eux dans le cluster choisi.
  • Coef cluster moyen : c'est la probabilité que deux voisins d'un sommet (site) choisi aléatoirement soient eux-mêmes voisins. (exemple : un sommet A possède 2 voisins B et C, le coefficient de clustering nous donne la probabilité que B et C sois aussi connecté entre eux c'est à dire que B et C possède aussi des mots clés communs).
Echelle : Si CCM proche de 0 => Les sites artistique du cluster choisi sont peu connecté entre eux. Si CCM proche de 1 => Les sites artistique du cluster choisi sont très connectés entre eux.
